# Learning Resources

Curated resources to supplement your AI learning journey.

## Free GPU Platforms

### Google Colab
- **Free Tier:** 12-15 hours of GPU access per day
- **GPU Types:** T4, P100 (free tier), A100/V100 (Colab Pro)
- **Best For:** Quick experiments, tutorials, sharing notebooks
- [Get Started →](https://colab.research.google.com/)

### Kaggle Notebooks
- **Free Tier:** 30 hours GPU per week, 20 hours TPU per week
- **GPU Types:** P100, T4
- **Best For:** Competitions, large datasets, longer training
- [Get Started →](https://www.kaggle.com/code)

### Comparison Table

| Feature | Google Colab Free | Kaggle Free | Colab Pro |
|---------|-------------------|-------------|-----------|
| GPU Hours | ~12hrs/day | 30hrs/week | ~24hrs/day |
| GPU Type | T4 | P100, T4 | A100, V100 |
| RAM | 12GB | 16GB | 32GB |
| Disk | 15GB | 20GB | 50GB |
| Cost | Free | Free | $10/month |
| Timeout | 12 hours | 9 hours | 24 hours |

## Datasets

### Public Dataset Repositories
- **Kaggle Datasets:** [Browse](https://www.kaggle.com/datasets) - Thousands of datasets with notebooks
- **UCI ML Repository:** [Browse](https://archive.ics.uci.edu/ml/) - Classic ML datasets
- **HuggingFace Datasets:** [Browse](https://huggingface.co/datasets) - NLP and multimodal datasets
- **Google Dataset Search:** [Search](https://datasetsearch.research.google.com/) - Find datasets across the web

### Recommended Datasets for Practice
- **MNIST:** Handwritten digits (beginner-friendly)
- **CIFAR-10/100:** Object classification
- **ImageNet:** Large-scale image classification
- **COCO:** Object detection and segmentation
- **IMDb Reviews:** Sentiment analysis
- **WikiText:** Language modeling
